## 变化规则

规则的一般过去时动词变位以一般现在时的==第三人称单数==为基础进行变位。

以fragen为例：

| 人称 | 现在时 (Präsens) | 一般过去时 (Präteritum) |
| ---- | ---------------- | ----------------------- |
| ich  | frage            | fragt==e==              |
| du   | fragst           | fragt==est==            |
| er   | ==**fragt**==    | fragt==e==              |
| wir  | fragen           | fragt==en==             |
| ihr  | fragt            | fragt==et==             |
| sie  | fragen           | fragt==en==             |

以arbeit为例：

| 人称      | 现在时 (Präsens) | 一般过去时 (Präteritum) |
| --------- | ---------------- | ----------------------- |
| ich       | arbeite          | arbeitet==e==           |
| du        | arbeitest        | arbeitet==est==         |
| er/sie/es | ==**arbeitet**== | arbeitet==e==           |
| wir       | arbeiten         | arbeitet==en==          |
| ihr       | arbeitet         | arbeitet==et==          |
| sie/Sie   | arbeiten         | arbeitet==en==          |

### 变为 u

| 动词 (Infinitiv) | 第一人称过去式 (Präteritum) |
| ---------------- | --------------------------- |
| backen           | buk (或 backte)             |
| tragen           | trug                        |
| fahren           | fuhr                        |
| wissen           | wusste                      |
| schlagen         | schlug                      |
| waschen          | wusch                       |
| werden           | wurde                       |
| einladen         | lud ein                     |

> 在一个阳光明媚的早晨，安娜决定烤（backen）一个蛋糕，并邀请（einladen）邻居们来品尝。她穿上围裙，弟弟马克帮忙搬运（tragen）食材，爸爸开车（fahren）去超市买了一些新鲜水果。“你知道（wissen）蛋糕要烤多久吗？”马克好奇地问。安娜笑着回答：“当然知道，大概40分钟。”蛋糕出炉后，安娜小心地切（schlagen）下一块，递给马克。他尝了一口，赞叹道：“太棒了！”吃完蛋糕，安娜开始洗（waschen）餐具，马克则帮忙整理桌子。妈妈笑着说道：“你们真是越来越懂事，将来一定会成为（werden）很棒的大人。”邻居们陆续到来，大家一起分享蛋糕，欢声笑语充满了整个屋子，安娜和家人都感到无比幸福。
