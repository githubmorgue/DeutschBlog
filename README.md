这个仓库目前主要为语料语块积累笔记，其目的和作用有二，一是整理各个模块文章中的重难点，二是积累其中可以用于自己口语和书面的表达方式以便考试复习。

这个仓库知识点的来源主要为sharplingo(<a href="https://www.sharplingo.cn/">https://www.sharplingo.cn/</a>)。区别于scharplingo的网站知识，我针对文章的难点和知识点进行了提炼、整理和总结，更加有条理的呈现知识，非常合适作为学习后的补充阅读；同时网站内容为知识密集型的干货，避免被大量的文本淹没，适合备考。

文章整理的知识点基于作者自己当时的想法，难免主观。所以我想该仓库的笔记对读者来说仅是参考，更建议读者在学习过程中也建立自己的笔记仓库。如果在阅读过程中发现纰漏和补充也可邮件（<a href="morguehu@gmail.com">morguehu@gmail.com</a>，更推荐）或文末留言，我收到消息后会处理。
